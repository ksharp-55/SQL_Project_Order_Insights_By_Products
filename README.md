# SQL_Project_Order_Insights_By_Products
SQL analysis of order volumes and product category to identify sales trends.


## Order and Product category analysis (SQL, Python)

## Overview
This project analyzes customer orders and product category to identify sales patterns and demand trends. SQL and Python are used together for accurate data analysis.


## Key Objectives
-Analyze order volume by product codes
-Evaluate product size distribution
-Identify sales and demand trends
-Support data-driven decisions


## Tools and Technologies
-SQL for data extraction and queries
-Python for data analysis and logic
-pandas and numpy for processing
-Visualizaiton libraries


## Description
This project combines SQL and Python to analyze order data and product codes. SQL queries extract structured data, while Python preocesses and analyzes the results to uncover insights about sales behavior and demand.


## Key Insights
-High and low demand products
-Order size distribution patterns
-Product performance trends
-Data-supported business insights


## How to Use
1. Run SQL queries to extract data
2. Load results into Python
3. Perform analysis using Python
4. Review trends and insights
